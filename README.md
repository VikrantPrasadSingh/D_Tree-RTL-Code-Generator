# Decision Tree RTL Code Generator

A Python-to-Verilog code generator that automatically converts a trained sklearn Decision Tree into synthesisable RTL hardware. Applied to breast cancer classification using the Wisconsin Breast Cancer Dataset.

---

## What This Project Does

Most ML models run on CPUs or GPUs. This project takes a trained Decision Tree and converts it into pure combinational Verilog logic — enabling inference on bare hardware with no processor required.

The Python codegen script reads the tree's internal structure (thresholds, feature indices, leaf predictions) and automatically writes a `.v` file with nested if-else comparators that perfectly mirror the trained model.

```
            ↓

    (depth=4, criterion=entropy)

    Python Codegen Script
    (reads tree_.feature, tree_.threshold, tree_.children_left/right)
            ↓
    Generated Verilog RTL
    (combinational if-else logic, 8-bit fixed point inputs)
            ↓
    Icarus Verilog Simulation
    (testbench verifies 114 test samples against Python predictions)
```

---

## Key Features

- **Automatic RTL generation** — codegen script works on any sklearn Decision Tree, not just this dataset
- **Fixed-point arithmetic** — floating point thresholds scaled to 8-bit integers (multiply by 255), eliminating floating point hardware overhead
- **Pure combinational logic** — no clock required, inference completes in a single propagation delay
- **End-to-end verified** — hardware predictions cross-checked against Python golden reference across all test samples
- **< 1% accuracy loss** — fixed-point quantisation introduces minimal degradation from software baseline

## Dataset

Wisconsin Breast Cancer Dataset — built into sklearn (`load_breast_cancer()`).

- 569 samples, 30 features, 2 classes
- Features: cell nucleus measurements (radius, texture, perimeter, area, concavity etc.)
- Classes: 0 = Malignant, 1 = Benign
- No missing values, no duplicates

---

## EDA Insights

- **Most important feature** — mean concave points (strongest predictor, aligns with medical knowledge that malignant cells have irregular jagged boundaries)
- **Top correlated features with target** — worst concave points, worst perimeter, worst radius
- **Class distribution** — 357 benign, 212 malignant (slightly imbalanced, no resampling needed)

```verilog
    input  [7:0] worst_radius,

        if (mean_concave_points <= 8'd64) begin
            if (worst_radius <= 8'd75) begin
                predicted_class = 1'b1;
            end else begin
                // ...
            end
        end else begin
            predicted_class = 1'b0;
        end
    end
endmodule
```
